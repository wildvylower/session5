# Student Club Hub

A premium, interactive web portal designed for school communities to discover, explore, and join student-led clubs.

## Tech Stack

*   **React** - Frontend library for rendering interactive components.
*   **Vite** - Development server and asset compiler.
*   **Tailwind CSS v4** - Styling framework for custom UI layouts and glowing dark-theme aesthetics.
*   **React Router v7** - Declarative routing engine for single-page views.
*   **React Icons (FontAwesome)** - Unified svg iconography.
*   **Local Storage API** - Browser persistence to store club registration status.

## Features

1.  **Homepage** - Immersive, responsive landing layout with visual showcases.
2.  **Clubs Directory** - Search and explore student organizations, equipped with name sorting (A-Z/Z-A) and display mode toggles (Grid/List View).
3.  **Details Page** - Multi-column overview for each club containing descriptions, schedules, and active join membership toggles.
4.  **Local Persistence** - Real-time client-side joining logic stored in the browser.
5.  **Custom 404 Page** - Navigation fallback to gracefully guide lost routes back home.

## Installation & Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the local development server:
    ```bash
    npm run dev
    ```

3.  Build the bundle for production hosting:
    ```bash
    npm run build
    ```
